# Java 14
## Records
Records modülü ile gereksiz kod kalabalığı azaltıldı.
```java
//Java 14 den önce
final class Rectangle{
    public final int w;
    public final int h;

    public Rectangle(int w, int h) {
        this.w = w;
        this.h = h;
    }
}
```
```java
//Java 14
record Rectangle(int w, int h){}
```
## InstanceOf Cast
```java
//Java 14 den önce
if (obj instanceof Car) {
    Car car = (String) obj;
	car.start();
}
```
```java
//Java 14
if (obj instanceof Car car) {
	car.start();
}
```
